# Pathfinding Algorithm Visualizer

An interactive web-based tool designed to visualize classic graph search algorithms and automated maze generation patterns in real-time. 

## Planned Features & Architecture
* **Visualizations**: Real-time animations for BFS, DFS, and A* Search.
* **Maze Generation**: Automated test grids utilizing Recursive Division.
* **Tech Stack**: Vanilla JavaScript, HTML5 Canvas/DOM, CSS3.

## Current Status: In Active Development
* Initialize 2D grid matrix mapping system.
* Implement BFS & DFS search logic.
* Integrate heuristic functions for A* Search.
* Add dynamic CSS cell-state animations.
